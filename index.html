2D Plotter & Laser Cutter

Mechanical Design Arduino CAD Laser Systems

Overview

Converted a 3D printer into a versatile 2D plotting and laser cutting system. This project demonstrates the flexibility of repurposing existing hardware for new applications, combining mechanical engineering with electronics and software.

Key Features

  • Custom firmware modifications for 2D operation modes
  • Specialized tool heads for both plotting and laser cutting
  • Precision control system for accurate line work and cutting
  • Safety features including emergency stops and laser protection

Applications

The system has been used for creating technical drawings, producing artwork, and performing precision cutting on various materials. It serves as both a creative tool and a practical fabrication device.
